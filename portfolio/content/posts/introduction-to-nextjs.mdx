# Einführung in Next.js

Next.js ist ein leistungsstarkes React-Framework, das serverseitiges Rendering (SSR) und statische Seitengenerierung (SSG) unterstützt. Es bietet eine Vielzahl von Funktionen, die die Entwicklung von React-Anwendungen vereinfachen und beschleunigen.

## Was ist Next.js?

Next.js wurde von Vercel entwickelt und bietet eine Reihe von Funktionen, die über das hinausgehen, was mit Create React App möglich ist. Dazu gehören:

- **Serverseitiges Rendering (SSR):** Generiert Seiten dynamisch auf dem Server.
- **Statische Seitengenerierung (SSG):** Generiert Seiten zur Build-Zeit.
- **API-Routen:** Erstellen Sie serverseitige Endpunkte innerhalb Ihrer Next.js-Anwendung.
- **Bildoptimierung:** Automatische Optimierung von Bildern mit der `next/image`-Komponente.
- **Internationalisierung (i18n):** Unterstützung für mehrere Sprachen und Regionen.

## Vorteile von Next.js

Next.js bietet mehrere Vorteile gegenüber herkömmlichen React-Anwendungen:

- **SEO-freundlich:** Durch serverseitiges Rendering und statische Seitengenerierung verbessert Next.js die SEO-Leistung Ihrer Anwendung.
- **Schnelle Ladezeiten:** Statische Seiten werden zur Build-Zeit generiert und können schnell ausgeliefert werden.
- **Einfache API-Integration:** Mit API-Routen können Sie serverseitige Endpunkte innerhalb Ihrer Anwendung erstellen.
- **Bildoptimierung:** Die `next/image`-Komponente optimiert Bilder automatisch für verschiedene Bildschirmgrößen und Auflösungen.

## Beispiel für eine Next.js-Seite

Hier ist ein einfaches Beispiel für eine Next.js-Seite, die serverseitiges Rendering verwendet:

```jsx
import { GetServerSideProps } from 'next'

export default function Home({ data }) {
  return (
    <div>
      <h1>Willkommen zu meiner Next.js-Seite</h1>
      <p>Data: {data}</p>
    </div>
  )
}

export const getServerSideProps: GetServerSideProps = async (context) => {
  // Daten vom Server abrufen
  const res = await fetch('https://api.example.com/data')
  const data = await res.json()

  return {
    props: {
      data,
    },
  }
}
